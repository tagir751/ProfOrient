# mTracker — Система трекинга прогресса учеников

## 🚀 Установка на Beget (mtrecker.tagir75.ru)

**Подробная инструкция:** см. файл [INSTALL_BEGET.md](./INSTALL_BEGET.md)

### Краткая инструкция:

1. **Подготовка:**
   - Установите Node.js 20.x в `~/.local` (см. INSTALL_BEGET.md)
   - Откройте доступ к `~/.local` через панель Beget
   - Создайте папки `data/`, `tmp/`, `logs/` с правами 755

2. **Загрузка:**
   ```bash
   cd ~/mtrecker.tagir75.ru/public_html
   # Загрузите файлы через FTP или Git
   ```

3. **Настройка .htaccess:**
   - Замените `tagir7ow` на ваш логин Beget в файле `.htaccess`

4. **Установка зависимостей:**
   ```bash
   npm ci --production
   ```

5. **Запуск:**
   ```bash
   touch tmp/restart.txt
   ```

6. **Первый вход:**
   - Логин: `admin`
   - Пароль: `admin123`
   
   **Сразу смените пароль!**

## Функционал

### Администратор
- Управление всеми пользователями
- Создание/редактирование/удаление
- Смена паролей пользователям
- Экспорт всей базы в Excel
- Импорт базы из Excel (с предпросмотром)

### Наставник
- Добавление своих учеников
- Просмотр прогресса учеников
- Добавление метрик
- Редактирование своих учеников

### Ученик
- Ведение сессий
- Добавление действий
- Просмотр проектов
- Графики прогресса
- Печать карточки

## Цветовая схема "Морская волна"
- Тёмно-океанский: #006994
- Океанский синий: #0077be
- Бирюзовый: #40e0d0
- Небесно-голубой: #87ceeb
- Облачный белый: #f0f8ff

## Безопасность
- Пароли хранятся в хешированном виде (bcrypt)
- Минимальная длина пароля: 4 символа
- Пароль должен содержать буквы и цифры
- История изменений сохраняется

## Экспорт/Импорт
- Полный экспорт всех данных в Excel
- Карточка ученика для печати (HTML)
- Импорт с выбором режима (замена/добавление)
- Предпросмотр перед импортом
