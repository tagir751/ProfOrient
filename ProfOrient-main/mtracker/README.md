# mTracker — Система трекинга прогресса учеников

## Установка на Beget

### 1. Загрузка файлов
Загрузите все файлы проекта в папку `/home/tagir7ow/tagir/public_html/`

### 2. Установка зависимостей
В панели управления Beget перейдите в раздел "Node.js" и нажмите "Установить зависимости" или выполните:
```bash
cd /home/tagir7ow/tagir/public_html
npm install
```

### 3. Настройка
В панели управления Beget:
- Перейдите в раздел "Node.js"
- Укажите файл запуска: `app.js`
- Нажмите "Запустить"

### 4. Первый вход
- Логин: `admin`

## Функционал

### Администратор
- Управление всеми пользователями
- Создание/редактирование/удаление
- Смена паролей пользователям
- Экспорт всей базы в Excel
- Импорт базы из Excel (с предпросмотром)

### Наставник
- Добавление своих учеников
- Просмотр прогресса учеников
- Добавление метрик
- Редактирование своих учеников

### Ученик
- Ведение сессий
- Добавление действий
- Просмотр проектов
- Графики прогресса
- Печать карточки

## Цветовая схема "Морская волна"
- Тёмно-океанский: #006994
- Океанский синий: #0077be
- Бирюзовый: #40e0d0
- Небесно-голубой: #87ceeb
- Облачный белый: #f0f8ff

## Безопасность
- Пароли хранятся в хешированном виде (bcrypt)
- Минимальная длина пароля: 4 символа
- Пароль должен содержать буквы и цифры
- История изменений сохраняется

## Экспорт/Импорт
- Полный экспорт всех данных в Excel
- Карточка ученика для печати (HTML)
- Импорт с выбором режима (замена/добавление)
- Предпросмотр перед импортом
